# ☽ Daily Wisdom — Islamic & Motivational Quote Generator

A beautiful, lightweight random quote generator featuring **Islamic (Quran & Hadith)** and **Motivational** quotes. Built with pure HTML, CSS & JavaScript — no frameworks, no dependencies.

---

## ✨ Features

- 📖 **24+ Quotes** — 12 Islamic (Quran & Hadith) + 12 Motivational
- 🔤 **Arabic Text** — Quranic ayaat displayed in Arabic with source reference
- 🎨 **Clean UI** — White background with purple theme, smooth animations
- 🔀 **Random Generator** — New quote on every click, no repeats
- 🏷️ **Category Filter** — Filter by All / Islamic / Motivational
- 📋 **Copy Button** — One click to copy any quote
- 📱 **Fully Responsive** — Works on mobile, tablet, and desktop
- 📤 **Share Buttons** — Share directly to WhatsApp, Facebook & Twitter/X

---
